## 🚀 Quick Start: Enable Live News Updates

### Get NewsAPI Key (2 minutes):
1. Visit: https://newsapi.org/register
2. Sign up (free - 100 requests/day)
3. Copy your API key

### Add to Project:
1. Create `.env` file in project root
2. Add this line:
   ```
   VITE_NEWS_API_KEY=paste_your_key_here
   ```
3. Restart dev server: `npm run dev`

### Done! 
Your notification bell now shows live tax & compliance news from India.

---

**Note:** The `.env` file is already in `.gitignore` - your API key won't be committed to git.

For detailed instructions, see `NEWS_API_SETUP.md`
